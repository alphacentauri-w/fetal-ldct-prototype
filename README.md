# Прототип генерации синтетических НДКТ-подобных изображений плода по МРТ

Проект является локальным прототипом для магистерской ВКР. Он демонстрирует полный путь обработки одного МРТ-среза или выбранного среза NIfTI-томограммы: загрузка изображения/томограммы, предобработка, генерация КТ-подобного изображения без обучения GAN, имитация низкодозовой КТ и расчет базовых метрик качества.

> ⚠️ Прототип не является медицинским изделием и не предназначен для клинической диагностики. Модуль `gan_module.py` содержит детерминированное преобразование интенсивностей, которое в будущем можно заменить настоящей GAN-моделью.

## Структура проекта

```text
.
├── app.py                # Streamlit-интерфейс
├── preprocessing.py      # Предобработка МРТ-срезов
├── gan_module.py         # Заменяемый модуль генерации синтетического КТ
├── ldct_simulator.py     # Имитация низкодозовой КТ
├── metrics.py            # MSE, PSNR, SSIM
├── requirements.txt      # Python-зависимости
├── README.md             # Инструкция и описание
└── outputs/              # Папка для сохранения результатов
```

## Возможности

- загрузка МРТ-среза в формате PNG, JPG или JPEG, а также 3D-томограмм NIfTI в формате `.nii` и `.nii.gz`;
- выбор среза NIfTI-томограммы через slider с отображением размерности загруженного объема;
- перевод изображения в оттенки серого;
- нормализация интенсивностей в диапазон `[0, 1]`;
- изменение размера до `256 × 256`;
- генерация синтетического КТ-подобного изображения;
- имитация низкодозовой КТ с шумом Пуассона, гауссовым шумом, снижением контрастности и небольшим гауссовым размытием;
- расчет MSE, PSNR и SSIM;
- сохранение изображений и таблицы метрик в `outputs/`.

## Установка

Рекомендуется использовать виртуальное окружение Python 3.10+.

```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

После запуска откройте локальный адрес, показанный Streamlit в терминале, загрузите PNG/JPG МРТ-срез или NIfTI-томограмму (`.nii`, `.nii.gz`) и при необходимости сохраните результаты кнопкой **«Сохранить результаты в outputs»**. Для NIfTI приложение покажет размерность 3D-томограммы и предложит выбрать номер 2D-среза через slider; выбранный срез будет передан в тот же прототипный pipeline, что и PNG/JPG.

## Замена прототипного генератора на GAN

Функция `generate_synthetic_ct(image)` в `gan_module.py` является стабильной точкой интеграции. В будущей версии вместо текущего преобразования интенсивностей можно загрузить обученную модель и вернуть результат инференса в виде `numpy.ndarray` с тем же диапазоном значений `[0, 1]`.
